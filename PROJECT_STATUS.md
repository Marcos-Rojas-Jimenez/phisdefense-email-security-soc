# Estado del proyecto

## Estado actual

Este repositorio contiene una versión preparada para portfolio profesional del proyecto PhisDefense SOC & Email Security Lab.

El servidor original del TFM sigue activo en el momento de generación de este paquete, pero este repositorio está diseñado para conservar el valor del proyecto aunque el servidor deje de estar disponible.

- 270 pruebas totales agregadas en el dashboard.

- 3 eventos spoofing_reject registrados.
- Dataset SOC público sanitizado.
- Documentación técnica preparada para portfolio.
